Как запустить/проверить:
- для запуска main.py из строки терминала ввести: python3 main.py
- для просмотра файлов, отслеживаемых GIT,: git ls-files
- для добавления в отслеживание: git add . или git add [имя файла] 
- для создания коммитов: git commit -m "add Readme.md"
- для исключения venv: touch .gitignore и добавляем venv/
